# Reporting Outputs

## Public Summary Report (end-of-grant)

**Sections:**
- Executive summary (scope, targets, primary invariant)
- Corpus statistics (counts by category and format)
- Divergence classes (counts + representative examples)
- Canonical mismatch findings (by format)
- Spec feedback summary (issues raised and status)
- Recommendations for implementers

## Maintainer Brief

**Audience:** implementer maintainers + spec editors

**Contents:**
- Top divergence cases with minimal repros
- Severity notes (crash vs mismatch vs ambiguity)
- Proposed spec clarifications
- Regression tests added

## Required Emphasis

- Highlight primary invariant failures and how they were resolved or tracked.
- Include DRISL1 and CID/CAR coverage summaries.
